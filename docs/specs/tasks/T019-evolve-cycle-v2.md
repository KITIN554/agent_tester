# T019 — Запустить evolve cycle, получить v2

## Приоритет: КРИТИЧНО (#4, ~30 мин)

## Цель
После T018 (улучшенный генератор) и T017 (улучшенные промпты агентов)
прогнать полный эволюционный цикл и убедиться, что оба механизма работают
вместе: генератор не отбраковывает 50% кандидатов, цикл показывает
lead time, прогон v2 не хуже v1.

## Что нужно сделать

```bash
set -a && . ./.env && set +a

tester evolve cycle --system finance_agent --rounds 1 --target-count 3
tester evolve cycle --system travel_agent  --rounds 1 --target-count 3
```

Проверить:
1. Все 3 кандидата для каждой корзины валидны (или отбракован максимум 1).
2. Прогон в составе цикла отработал без exit=3.
3. `lead_time_metrics` записаны в `report.json`.
4. Сгенерированные сценарии действительно покрывают новый угол —
   сравнить описания с уже существующими.

## Acceptance criteria

- [ ] Оба `evolve cycle` отработали с exit=0 (или 1, 2 — но не 3).
- [ ] В `baskets/finance_agent/` и `baskets/travel_agent/` появились
  новые YAML (минимум 2 на корзину).
- [ ] `lead_time_metrics` в обоих новых `report.json` непустые.
- [ ] Закоммичено: `chore: v2 evolution cycle (auto-generated scenarios + run + analysis)`

## Зависимости
T017 (v1 baseline), T018 (улучшенный генератор).

## Бюджет
Ожидаемая стоимость: $0.20–0.50 за обе корзины; время: 5–10 мин.
